Export / Import

Learning goals:

Can work with both types of import/export in HTML.
  • You can use the script in html like this:
    • <script src="./pages/lektioner/lektioner.js"></script>
    Can import/export in both ways in Node.js (CommonJS and ES Modules). Knows the pros of using ES Modules.
  • CommonJS:const fs = require("fs"); module.exports = myFunction;
  • ES Modules:import fs from "fs"; export default myFunction;
  • Pros of ES Modules:
  • It is now the standard in JavaScript
  • Top level await support
    • Understands the security need for serving static files and can do it in Express.
  • You can serve static files in express by using the built in middleware function express.static.
  • Example: app.use(express.static('public')); This will serve all files in the public folder.
    • Can recount the number of ways and approaches to redirection.
  • You can do it in express with this: res.redirect("/newpage");
  • Also in Html (client side) with: window.location.href = "/newpage";
    • Understands the difference between client-side redirection and server-side.
