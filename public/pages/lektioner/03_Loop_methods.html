    • Can use loop methods (+map, +filter, %reduce). Can choose the right one at the appropriate time.
  • .forEach() runs a function for each element and doesn't return a value.
  • .map() creates a new array by transforming each element.
  • .filter() returns a new array with elements that pass a test.
  • .reduce() combines all elements into a single value (sum, object, etc.).
  • .sort() sorts elements in place (modifies the original array).
  • These are just some of them and you can read more about them on w3schools for example
